# Kcuni BotFather version

## Render deployment

The root `render.yaml` deploys this bot as a free Render web service. Render
provides `RENDER_EXTERNAL_URL`; when it is present, Kcuni registers
`/telegram/webhook` with Telegram and listens on `PORT`. Without a public URL,
the same entry point uses long polling for local development.

Required Render secret: `BOT_TOKEN`. Optional AI secret: `GEMINI_API_KEY`.
Never store either value in this repository.

Это простая версия Kcuni как обычного Telegram-бота через BotFather.

Плюсы:

- не нужен отдельный номер телефона;
- бот выглядит как `@YourBotName`;
- стиль меняется с телефона командой `/still`;
- есть память по каждому пользователю;
- есть команда `/news`;
- можно подключить OpenAI-compatible API или оставить демо-режим.

## 1. Создать бота

1. Открой Telegram.
2. Найди `@BotFather`.
3. Напиши `/newbot`.
4. Создай имя и username.
5. Скопируй токен.

## 2. Настроить

Скопируй `.env.example` в `.env` и вставь токен:

```env
BOT_TOKEN=123456:your_token_here
```

Если хочешь ответы через AI, добавь:

```env
OPENAI_API_KEY=your_key_here
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=gpt-4.1-mini
```

## 3. Запустить

```bash
npm install
npm start
```

## Команды

- `/start` - приветствие
- `/help` - список команд
- `/still` - показать стили
- `/still cute` - няшная
- `/still calm` - спокойная
- `/still playful` - игривая/дерзкая
- `/still serious` - серьёзная
- `/cute`, `/calm`, `/playful`, `/serious` - быстро переключить стиль
- `/news` - короткая выжимка новостей
- `/news week` или `/week` - сводка новостей за последние 7 дней
- `/new` - то же самое, короткая команда
- `/topics` - выбрать тип новостей
- `/web запрос` - поиск информации в интернете
- `/search запрос` - то же самое, явная команда поиска
- `/url https://site.com/page` - открыть и кратко пересказать конкретную страницу
- `/proactive` - включить/выключить сообщения, которые Kcuni пишет сама
- `/schedule 13:00,21:00,23:30` - настроить время самостоятельных сообщений
- `/timezone Europe/Minsk` или `/timezone +3` - настроить местное время
- `/location Минск` - запомнить город и известный часовой пояс
- `/stickers` - проверить, сколько стикеров она запомнила
- `/memory` - что Kcuni помнит о тебе
- `/forget` - очистить память

## Стикеры

Просто отправь Kcuni любой стикер. Она сохранит его `file_id` и сможет иногда отвечать им.

Стикер в Telegram отправляется отдельным сообщением. Внутрь текстового сообщения его вставить нельзя, но Kcuni может
сначала написать текст, а рядом отправить стикер.

Стикеры отправляются редко и по контексту: приветствие, смешной момент, нежная фраза, мат/эмоция, реакция на фото или
голосовое. Если контекст обычный, стикер не отправляется.

## Сообщения от неё самой

По умолчанию Kcuni может сама писать пользователям, которые уже нажали `/start`. Расписание задаётся в `.env`:

```env
DEFAULT_TIMEZONE=Europe/Minsk
PROACTIVE_SCHEDULE=13:00,21:00,23:30
PROACTIVE_CHECK_MINUTES=5
PROACTIVE_SILENCE_MINUTES=360
MAX_MESSAGES_PER_REPLY=2
```

Команда `/proactive` включает или выключает это для конкретного чата.

Kcuni может сама приносить новости без `/news`: в один из запланированных слотов она иногда выбирает что-то из заданной
темы и пишет естественно, не как новостной бот.

Самостоятельные сообщения отправляются по локальному расписанию пользователя. По умолчанию это `13:00`, `21:00` и
`23:30` в `Europe/Minsk`, если перед этим в чате была достаточно длинная пауза. Расписание и пояс можно менять командами
`/schedule` и `/timezone`. На бесплатном Render сервис может засыпать, поэтому сообщение по расписанию иногда придёт позже,
когда сервис снова проснётся.

## Память разговоров

Kcuni хранит недавние реплики с ролями и автоматически строит краткие итоги по дням, неделям и месяцам. Эти выжимки
попадают в контекст новых ответов, поэтому она может естественно вспомнить недавнюю тему без загрузки всей истории.
Команда `/memory` показывает текущую выжимку, а `/forget` удаляет журнал и все итоги.

На бесплатном Render файловая система временная: память переживает обычную работу процесса, но может сброситься после
перезапуска или нового деплоя. Для гарантированного долговременного хранения позже стоит подключить внешнюю базу данных.

## Понимание стикеров

Статические WebP-стикеры анализируются через Gemini Vision. Для анимированных и видео-стикеров используется превью.
Kcuni отвечает реакцией по смыслу картинки, а не общей фразой, и может показать неприязнь к неприятному стикеру.

Темы новостей:

```text
/topics interesting
/topics sad
/topics war
/topics tech
/topics mixed
```

## Как сделать ответы реально умными

Без `OPENAI_API_KEY` бот работает на локальных правилах. Это лучше, чем пустой бот, но это не настоящий разговорный AI.

Чтобы она отвечала как нормальная ИИшка:

1. Получи OpenAI-compatible API key.
2. Вставь в `.env`:

```env
OPENAI_API_KEY=your_key_here
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=gpt-4.1-mini
OPENAI_VISION_MODEL=gpt-4.1-mini
OPENAI_TRANSCRIBE_MODEL=whisper-1
```

3. Перезапусти:

```bash
npm start
```

Можно использовать не только OpenAI, а любой совместимый endpoint, если он поддерживает `/chat/completions`.

## Фото, голосовые и кружки

Kcuni умеет принимать:

- фото;
- голосовые;
- видео-кружки;
- стикеры.

Без AI-ключа она только замечает такие сообщения. Чтобы реально понимать фото и расшифровывать голосовые, добавь:

```env
OPENAI_API_KEY=your_key
OPENAI_VISION_MODEL=gpt-4.1-mini
OPENAI_TRANSCRIBE_MODEL=whisper-1
```

## Новости

В `.env` можно добавить RSS-источники через запятую:

```env
NEWS_FEEDS=https://example.com/rss,https://another.com/feed.xml
```

Обычный BotFather-бот не может сам читать чужие Telegram-каналы, если его туда не добавили.
Поэтому новости берутся из RSS/API или из каналов/групп, где бот реально состоит.

## Интернет-доступ

Kcuni может искать информацию и читать открытые страницы:

```text
/web что такое TDLib
/search последние новости AI
/url https://example.com/article
```

Ограничения обычные для серверного бота:

- не читает закрытые сайты без логина;
- не проходит CAPTCHA;
- не открывает платный/закрытый контент;
- не видит личные Telegram-чаты и закрытые каналы, если бота туда не добавили.
